Effortlessly Export WooCommerce Product Data to CSV with Versatile Filtering Options

== Description ==
This plugin is currently still in development and lacks many of the features listed below.

Export your WooCommerce products data to CSV using this free plugin.

Tailor your export precisely by selecting desired product data through our intuitive drag-and-drop interface.

Enjoy the simplicity of our user-friendly plugin interface, making product data export a breeze.

Additionally, schedule automatic exports on a daily, weekly, or monthly basis with ease using built-in timers.

== Installation ==
1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Start by visiting plugin settings at "WooCommerce > Step3 Report".
